# ФИНАЛЬНЫЙ ПРОЕКТ
   В проекте предсталены ответы к двум практическим заданиям:  
## Работа с базой данных 
   1. Запрос к базе данных. 
       а. На стенде приложения "Яндекс Самокат" было создано несколько заказов. 
       б. С помощью инструмента Postman были созданы курьеры, осуществленны принятие заказа, принятие заказа курьером, завершение заказа.  
       в. В консоли CygWin произведен  вход в бызу данных. С помощью операторов осуществленна проверка отображение заказа в базе данных  и убедиться, что статусы заказов записываются корректно в базе данных. 
       с. Ответ преставлен в виде скриншотов.
## Автоматизация теста к  API  
   1. В приложении VC code была создана функция выполняющая запрос на создание  заказа.
   2. Был сохранен трековый номер  заказа.
   3. Был создан автотест проверяющий получение заказа по номеру трекера заказа
### Шаги автотеста:
   1. В файле configuration.py изменить значение URL_SERVICE на актуальное и сохранить изменения.
   2. Перейти в файл test_samocat_order_track.py и запустить тест.
